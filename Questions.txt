Explain what the ".map" properties of arrays does.

Explain what the ".reduce" properties of arrays does.
It runs a callback function on each element of the array and returns a single value

How is ".map" different from reduce?
.map creates anew array, and .reduce takes all elements in array and makes them a single value thats returned
